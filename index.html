Why use a GitHub token?

  • Without token: 60 requests per hour
  • With token: 5,000 requests per hour
  • Access to private repositories (if token has permissions)
  • Better rate limiting and performance

How to create a GitHub token:

  1. Go to GitHub Settings ‚Üí Developer settings ‚Üí Personal access tokens
  2. Click "Generate new token (classic)"
  3. Give it a name (e.g., "Gitlify App")
  4. Select scopes: public_repo (or repo for private repos)
  5. Click "Generate token" and copy it
Your token is stored locally and never sent to any server except GitHub's API.
